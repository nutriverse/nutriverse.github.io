R Packages

Tools for Nutrition Research and Analysis

Our collection of packages developed by and contributed to by our community. These package were developed as part of a specific project but had more general use for common nutrition data analysis tasks. These packages are available for use by anyone needing to perform the same type of data analysis.


dietry: Utilities for Dietary Intake Indicators for Food Security Assessments

micronutr: Determining Vitamin and Mineral Status of Populations

nipnTK: National Information Platforms for Nutrition (NiPN) Data Quality Toolkit

riycf: Utilities for Calculating Infant and Young Child Feeding Indicators

sleacr: Simplified Lot Quality Assurance Sampling Evaluation of Access and Coverage (SLEAC) Tools in R

smartr: Tools for Nutritional Status and Mortality Rate Assessments and Surveys

squeacr: Semi-Quantitative Evaluation of Access and Coverage (SQUEAC) Tools in R

zscorer: Anthropometric z-score Calculator
